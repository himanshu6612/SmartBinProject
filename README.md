# Smart Bin AI ♻️

An AI-powered smart waste management system designed to improve waste segregation on university campuses.

## 🚀 Project Overview
This project uses computer vision and machine learning to classify waste items in real-time into:
- Recyclable
- Compost
- Landfill

The system provides instant feedback through a smart dashboard, helping reduce contamination in recycling streams.

## 🧠 Features
- Real-time waste classification using camera
- Vision Transformer (ViT) based model
- Interactive dashboard (Streamlit)
- Educational feedback for users
- Edge deployment ready

## 🛠️ Tech Stack
- Python
- PyTorch
- OpenCV
- Hugging Face (ViT)
- Streamlit
- TinyLlama (for AI explanations)

## 🎯 Problem Solved
Reduces 40–60% contamination in waste streams caused by improper segregation in campuses.
